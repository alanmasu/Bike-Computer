Detailed Description

GPS module header file.

This file contains the GPS module functions prototypes and some definitions. In hardware the connections are:

MSP432P401
-------------------
| |
RST -| |
| |
| |
| P3.3/UCA0TXD|----> GPS_L80 RX at 9600 8N1
-| |
| P3.2/UCA0RXD|----< GPS_L80 TX at 9600 8N1
| |
| |
-------------------
